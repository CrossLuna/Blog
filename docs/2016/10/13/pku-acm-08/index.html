本系列是整理2016年北京大學郭煒老師教授的ACM/ICPC暑期課投影片而成的筆記。
數學部分另外參考了 Victor Shoup 所著的 A Computational Introduction to Number Theory and Algebra

數論

整數基本性質

整個數論的中心觀念就是整數的可除性 (divisibility)。考慮整數集合 $\mathbb{Z} = {..., -2, -1, 0, 1, 2, ...}$ 中，有 $a, b \in \mathbb{Z}$ ，我們說 $a$ 可以整除$b$，如果存在 $z\in\mathbb{Z}$ 使得 $az = b$ ，記為 $a\mid b$ ，反之則記為 $a\nmid b$

定理 1.1 - 除法餘數性質

令 $a, b \in \mathbb{Z}$ 且 $b>0$ ，存在唯一一組 $q, r\in \mathbb{Z}$ 使得 $a = bq + r$ 且 $0 \leq r \le b$

證明

考慮一非負整數集合 $S$ ，其中元素皆形如 $a-bt, t\in\mathbb{Z}$ ，此集合顯然非空 ( Why? 如何製造它們? ) ，由 Well-ordering principle 1，我們知道這個集合有最小值，定義其為 $r$ 。僅由定義就可以滿足 $r = a-bq, q \in \mathbb{Z} $ 以及 $r\geq 0$ 兩條件，另外 $r < b$ 也必然成立，否則將會有 $r-b$ 也在 $S$ 中且比 $r$ 小，與之前 $r$ 為最小值的假設矛盾，事實上 如果 $r \geq b$ ，我們總是可以找到 $$0 \leq r-b = a - b(q+1)$$ 這個更小的值。
至此我們已證明存在性，接著證明唯一性，若存在 $r, q$ 以及 $r', q'$ 兩組整數對滿足條件，藉由消去 $a$ 我們可以推得 $$ r' - r = b(q - q') $$ 發現 $r'- r$ 是 $q'-q$ 的倍數，然而我們已經知道 $0\leq r, r' < b$ ，因此 $\left| r'-r \right| < b$，因此唯一可能的情況就是 $r' - r = 0$ ，由於 $b\neq 0$ ，亦可得到 $q-q' = 0$ 。

基本定理

1 Well-ordering principle: 任何非空的自然數子集，必定含有一個最小值，是一個由皮亞諾公理衍生的原理，無須證明，此原理與數學歸納法等價。
