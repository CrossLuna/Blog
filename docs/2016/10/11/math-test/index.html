這一章藉由一個複數編碼的問題為解題做一開端。

複數編碼問題

定義

若對於兩個具有整數虛部及實部的複數$X$與$B$，存在一組自然數$a_n, a_{n-1}, ..., a_1, a_0$，其中每個自然數都小於$|B|$，能滿足 $$ X = a_nB^n+a_{n-1}B^{n-1}+...+a_2B^2+a_1B+a_0 $$ 則我們稱這組自然數$a_n, a_{n-1}, ..., a_1, a_0$構成一對$(X, B)$的編碼。有了以上關係，我們可以使用$a_n, a_{n-1}, ..., a_1, a_0$來表示$X$，視其在一個以$B$為基底的數系上的投影。 我們的問題是，給定一對$(X, B)$，是不是能夠找到一組符合條件的$a_n, a_{n-1}, ..., a_1, a_0$對其進行編碼。

輸入: 為一個檔案，每行代表一組 $(X, B)$的四個整數，分別是$Re(X)$、$Im(X)$、$Re(B)$、$Im(B)$，其中$Re(X), Im(X) \leq 1000000$，$Re(B), Im(B) \leq 16$，如以下code.in所示
